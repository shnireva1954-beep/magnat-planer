## 2. Что исправлено в этой сессии (QA-аудит, уже в ветке)

Все правки в `app.html`, закоммичены и запушены:

1. **Убран горизонтальный скролл на вкладке «Привычки» при ширине ≤360px.**

3. **График «Доход vs Расход» стал двухцветным** (доход зелёный, расход красный) —
   раньше обе колонки были зелёные и не различались.

## 4. Робокасса — как заполнять анкету магазина

Значения для полей (partner.robokassa.ru → создать магазин):

| Поле | Что вписывать |
|------|---------------|
| Наименование магазина * | `Магнат` |
| Идентификатор магазина * | `magnatplaner` (латиницей; это будущий `MerchantLogin` в ссылке оплаты — потом менять тяжело, запиши его) |
| Ссылка на ваш ресурс * | `https://magnat-planer.ru` |
| Тип бизнеса (для оферты) | `Купля-продажа` (продаёшь готовый цифровой товар) |
| Способ вывода средств | Свой счёт/карта самозанятого (личные данные — заполняешь сам) |
| Адрес для корреспонденции | Необязательно (можно город: Краснокаменск) |

Важно:
- **«Скачать оферту» не нажимать** — своя оферта уже есть на сайте
  (`https://magnat-planer.ru/oferta.html`). Робокасса сама пишет: «если оферта есть — всё в порядке».
- Перед подачей заявки нужно, чтобы **способ вывода средств был добавлен и активирован**
  (об этом предупреждает красная строка в окне подтверждения).
- После подачи заявки данные магазина замораживаются — менять можно только
  технические настройки. Поэтому наименование/идентификатор/ссылку задать финально.
- «Продолжить» = принятие договора Робокассы (это нормально). Ответ придёт на почту и SMS.
- Почта в анкете (`artemmogulenko@icloud.com`) — контакт для самой Робокассы, это ок;
  публичная поддержка для клиентов на сайте остаётся `planermagnat@gmail.com`.

## 5. Самозанятый (НПД) — правила приёма платежей

Товар «Магнат» **подходит** под самозанятость (это свой цифровой продукт, а не
перепродажа чужого — перепродажа запрещена, но к нам не относится).

- **Лимит дохода: 2,4 млн ₽/год** нарастающим итогом. При 990 ₽ ≈ 2400 продаж/год.
  Превысишь — статус НПД слетает, нужно ИП.
- **Налог: 4%** с продаж физлицам (наши покупатели — физлица). Бонус 10 000 ₽ при
  регистрации гасит часть налога вначале. Платится раз в месяц до 28 числа.
- **Чек НПД обязателен по каждой продаже** (через «Мой налог»). Оферта п.3.2 обещает
  автоматический чек, поэтому **при активации магазина включи у Робокассы формирование
  чеков самозанятого** (связка с «Мой налог» / право регистрировать доход). Иначе чек
  придётся выбивать вручную по каждой оплате — за невыдачу штраф. **Обязательно проверить.**

## 6. ОБЯЗАТЕЛЬНО сделать до включения приёма денег (на стороне владельца)

1. **Запустить `SHEETS-BUILDER.gs`** по `SHEETS-INSTALL.md`, пройти чек-лист
   `SHEETS-SPEC.md`, сверить таблицы с превью на сайте. **Главный блокер.**
2. **Настроить письмо после оплаты** в ЛК Робокассы: ДВЕ ссылки — на таблицы и на
   `https://magnat-planer.ru/app.html`.
3. **Включить автоматические чеки самозанятого** (см. раздел 5).
4. Когда придёт платёжная ссылка/доступ Робокассы — вставить в `PAYMENT_URL`
   (см. раздел 8).

## 8. Когда Робокасса одобрит — что сделать с оплатой

В `index.html` найти строку:

```js
var PAYMENT_URL = "";
```

Вставить между кавычек платёжную ссылку Робокассы. После этого обе кнопки «Купить»
поведут прямо на оплату, а модалка-бронь отключится сама (логика уже готова,
`index.html`, блок «покупка»). Больше в коде ничего не нужно.

После публикации (`main` + push) сбросить OG-кэш Telegram отправкой URL боту @WebpageBot.
