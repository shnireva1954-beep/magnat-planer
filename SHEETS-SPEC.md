# Google-таблицы «Магнат 5 в 1» — продукт реализован

**Статус (июль 2026): таблицы реализованы кодом.** Генератор —
`sheets/magnat-builder.gs`, инструкция запуска — `sheets/README.md`.
Владелец запускает скрипт один раз и получает в своём Drive две папки:
демо-набор (для скриншотов и демо-ссылки) и чистый набор (для покупателей).

**Продавать можно после чек-листа** из `sheets/README.md` (шаг 4):
запуск, проверка с телефона, копия «как покупатель», скриншоты на лендинг.

## Архитектура каждого файла

| Лист | Назначение |
|---|---|
| **📊 Дашборд** | витрина: KPI-плитки, диаграммы, инсайт. Сетка скрыта, защита «предупреждение» |
| **Рабочие листы** | единственное место ввода: чекбоксы, списки, числа |
| **⚙️ Настройки** | параметры + инструкция «как настроить под себя» (≤5 строк) |
| *(скрытый)* **_calc** | все промежуточные формулы, чтобы дашборд был чистым |

Палитра превью: изумруд `#12a565`, золото `#c98a12`, красный `#e0556a`,
фиолетовый `#7a52d0`, синий `#2f8fd8`, шапки `#eef2f8`, сетка `#dfe6ef`.
Формулы в скрипте — en-US синтаксис (`setFormula`), Sheets локализует сам.

## Игровая механика (сверена с app.html)

- Монеты дня = `SUMPRODUCT(галочки; цены из Настроек)` (цены 10–30).
- День в зачёте при ≥3 привычках (порог — в Настройках).
- Серия — бегущий счётчик подряд зачтённых дней; множитель `MIN(2; 1+0.1·серия)`.
- Уровни по сумме монет: 0/300/1500/5000/15000 → 🐣→🚀→💼→🏢→👑.
- Прогресс уровня — SPARKLINE-бар.

## Что умеет каждый планер (реализовано)

1. **Привычки — «Штаб дисциплины»**: сетка месяца с чекбоксами и подсветкой
   «сегодня», heatmap % дня, монеты по дням; KPI серия/множитель/монеты/уровень/%
   месяца; кривая месяца (area), пончик месяца, сильные дни недели (columns),
   срез текущей недели, рекорд серии, лучшая/отстающая привычка, лист «История»
   с полуавтоматической строкой итогов месяца, инсайт.
2. **Задачи — «Центр задач»**: «Осталось» считается само (сегодня/завтра/−N дн.),
   просрочка краснеет, выполненные зачёркиваются, пилюли приоритетов;
   KPI открыто/горит/закрыто за 7 дн (с дельтой к прошлой неделе)/% в срок/
   средний возраст; блок «Сегодня в фокусе», пончик приоритетов, закрыто по
   дням недели, статистика потока, «Архив», инсайт со старейшей просрочкой.
3. **Тело — «Панель формы»**: дневник (вес·ккал·вода·сон·тренировка·группа
   мышц·настроение); KPI с дельтами; «Путь к цели» (старт→цель, % пути);
   график веса с линией цели; **прогноз даты достижения цели по реальному
   темпу** (SLOPE за 30 дней); радар «Баланс мышц»; тренировки недели;
   настроение недели (emoji-лента); инсайт про отстающую группу.
4. **Неделя — «Пульт недели»**: 7 блоков «3 главных дела + энергия 1–5 +
   настроение», фокус-цели; карточки дней, кривая энергии, пик/спад энергии,
   **сравнение с прошлой неделей** и спарклайн прошлых недель из архива
   «🧭 Рефлексия», инсайт «сложное — на пик, лёгкое — на спад».
5. **Финансы — «Капитал»**: журнал операций с категориями из Настроек;
   KPI доход/расход (с дельтами к прошлому месяцу)/сбережения (% дохода
   против цели)/подушка в месяцах/**прогноз расхода к концу месяца**;
   пончик категорий, план vs факт со SPARKLINE-барами и подсветкой
   перерасхода, копилка за 6 месяцев, топ-3 траты, долги с прогрессом, инсайт.

Пустые (чистые) версии не выглядят сломанными: вместо ошибок — подсказки
«внеси первую запись — аналитика оживёт».

## Принципы (не нарушать)

- Инсайты считаются ТОЛЬКО из реальных данных пользователя, никакой
  выдуманной статистики.
- Всё реализуемо штатными средствами Sheets — и реализовано ими
  (без макросов в самих таблицах: копия покупателя работает без разрешений).
- Ячейки ввода — единственные незащищённые; защита в режиме предупреждения.

## Осталось владельцу (вручную)

1. Запустить `buildAll` и `buildAllClean` по `sheets/README.md`.
2. Пройти чек-лист (телефон, копия покупателя, пустые состояния).
3. Скриншоты демо-дашбордов → заменить превью на лендинге, добавить ссылку
   «Открыть демо-таблицу» (view-only).
4. Ссылки чистых таблиц → в письмо после оплаты (ЛК Робокассы) вместе со
   ссылкой на app.html.
