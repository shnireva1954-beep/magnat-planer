# Как создать 5 реальных планеров «Магнат» (10 минут, без программирования)

Файл `magnat-builder.gs` — генератор продукта. Он сам создаёт в вашем
Google Drive пять готовых таблиц: дашборды, формулы, чекбоксы, диаграммы,
условное форматирование, защиту листов и демо-данные. Таблицы полностью
совпадают по стилю с превью на magnat-planer.ru.

## Шаг 1. Запуск (один раз)

1. Откройте [script.google.com](https://script.google.com) под аккаунтом,
   на котором будут храниться таблицы-оригиналы.
2. Нажмите **«Новый проект»**.
3. Удалите всё из файла `Code.gs` и вставьте туда **целиком** содержимое
   `magnat-builder.gs`. Нажмите 💾 (Сохранить).
4. Вверху, рядом с кнопкой «Выполнить», выберите в списке функцию
   **`buildAll`** и нажмите **«Выполнить»**.
5. Google спросит разрешения. Нажмите «Разрешить». Если появится экран
   «Приложение не проверено» — это нормально (скрипт ваш собственный):
   **Дополнительные настройки → Перейти на страницу «Без названия»**.
   Скрипт работает только в вашем Drive и никуда ничего не отправляет.
6. Подождите 2–4 минуты. Готово: в Drive появилась папка
   **«🏛 Магнат — планеры (демо)»** с пятью таблицами.

## Шаг 2. Две версии набора

| Функция | Что создаёт | Для чего |
|---|---|---|
| `buildAll` | 5 таблиц **с демо-данными** | скриншоты для лендинга, демо-таблица для покупателей «посмотреть» |
| `buildAllClean` | 5 **чистых** таблиц | то, что получает покупатель |

Запустите обе — получите две папки. Пустые таблицы не выглядят сломанными:
вместо цифр дашборд подсказывает, с чего начать.

## Шаг 3. Как отдавать покупателю

1. В папке «для покупателя» откройте каждую таблицу →
   **Настройки доступа → Все, у кого есть ссылка → Читатель**.
2. Эти 5 ссылок вставьте в письмо после оплаты (настраивается в ЛК Робокассы),
   вместе со ссылкой на приложение https://magnat-planer.ru/app.html.
3. В письме одной строкой: «Откройте таблицу → меню **Файл → Создать копию** —
   и она ваша навсегда». Копия сохраняет все формулы, диаграммы и защиту.

## Шаг 4. Проверка перед продажей (чек-лист)

- [ ] Открыть каждый дашборд с телефона в приложении «Google Таблицы» — читается ли.
- [ ] В демо-версии всё считается: серия, монеты, уровень, графики не пустые.
- [ ] В чистой версии нет ошибок #REF/#DIV — вместо цифр подсказки «внеси первую запись».
- [ ] Сделать копию как покупатель и убедиться, что копия работает.
- [ ] Снять скриншоты демо-дашбордов и заменить ими превью на лендинге
      (плюс дать ссылку «Открыть демо-таблицу» — сильнейший аргумент доверия).

## Шаг 5 (по желанию). Автодата закрытия задач

В планере задач дату «Закрыта» можно проставлять автоматически по галочке ✓:
откройте файл «Магнат · Задачи» → **Расширения → Apps Script** → вставьте
содержимое `bonus-onedit-tasks.gs`, сохраните. Простой триггер не требует
разрешений и наследуется копиями покупателей. Без него дата вводится рукой —
таблица полноценно работает и так.

## Что внутри каждого планера

| Файл | Идея | Фишки дашборда |
|---|---|---|
| **Магнат · Привычки** | штаб дисциплины, ядро игры | серия 🔥, множитель, монеты, уровень, кривая месяца, пончик, сильные дни, срез недели, инсайт |
| **Магнат · Задачи** | управление потоком дел | «Сегодня в фокусе», горит/в срок, скорость за неделю, закрыто по дням, возраст задач, инсайт |
| **Магнат · Тело** | путь к цели | путь к цели с % и прогнозом даты, график веса с линией цели, радар мышц, сон/ккал/вода, настроение недели |
| **Магнат · Неделя** | энергия и фокус | карточки 7 дней, кривая энергии, фокус-цели, сравнение с прошлой неделей, архив в «Рефлексии» |
| **Магнат · Финансы** | деньги под контролем | доход/расход/сбережения/подушка, план vs факт с барами, копилка 6 месяцев, топ-3 траты, прогноз расхода, долги |

Игровая механика (монеты → серия → множитель ×1.1…×2 → уровни
0/300/1500/5000/15000) сверена с приложением app.html.

## Если захочется что-то поменять

Источник истины — скрипт. Поменяйте текст/формулу/цвет в `magnat-builder.gs`
и запустите `buildAll` заново: он создаст новую папку с новыми файлами
(старые не трогает — их можно удалить вручную).

Для разработчика: `node sheets/mock-test.js sheets/magnat-builder.gs` —
быстрая проверка скрипта без Google (границы диапазонов, формулы, листы).
